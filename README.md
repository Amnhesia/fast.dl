# 📦 AntoVaer FastDL

Este repositorio contiene los archivos del servidor **FastDL** (Fast Download) utilizados para la descarga rápida de recursos personalizados en servidores de juego.

## ¿Qué es FastDL?

FastDL es un sistema que permite a los jugadores descargar mapas, sonidos, modelos y otros archivos personalizados del servidor de forma rápida y directa desde un servidor HTTP externo, sin saturar el servidor de juego.

## 🎮 Servidores incluidos

| Carpeta | Juego |
|---|---|
| `left4dead2/` | Left 4 Dead 2 |
| `nmrih/` | No More Room in Hell |
| `Minecraft/` | Minecraft |

## 📁 Estructura

```
fast.dl/
├── left4dead2/     # Archivos para el servidor de Left 4 Dead 2
├── nmrih/          # Archivos para el servidor de No More Room in Hell
├── Minecraft/      # Archivos para el servidor de Minecraft
└── index.html      # Página principal del FastDL
```

## ⚙️ Uso

Este repositorio sirve como almacenamiento de archivos para descarga rápida. Los archivos aquí alojados son descargados automáticamente por los clientes al conectarse al servidor correspondiente.
